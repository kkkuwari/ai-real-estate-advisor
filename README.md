# AI Real Estate Investment Advisor

## Project Aim
This project is an academic prototype for an AI-assisted system that helps assess real estate investment opportunities.  
It combines a machine learning backend with a web frontend to support prediction-driven decision guidance.

## Planned Architecture
- `backend/`: FastAPI service, SQLite persistence, and model-serving logic.
- `backend/ml/`: scripts for dataset generation, preprocessing, model training, and evaluation.
- `backend/data/`: CSV datasets and processed data artifacts.
- `backend/models/`: saved trained model files.
- `frontend/`: Next.js React application for user interaction and result visualization.
- `evaluation/`: exported metrics, experiment outputs, and testing results.
- `report_assets/`: screenshots, diagrams, and notes used in the final academic report.

## Current Status
Initial project scaffold is created.  
Implementation will be developed incrementally in subsequent steps.
